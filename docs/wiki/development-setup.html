Development Setup

Get started with PyRC development with this comprehensive setup guide.

Note: These instructions assume you're familiar with basic command-line usage and Python development.

Required Software

  • Python 3.8+ - Core programming language
  • Git - Version control system
  • pip - Python package installer
  • Virtual Environment - Dependency isolation

Recommended Tools

  • VS Code or PyCharm - Code editors
  • Poetry - Dependency management
  • Docker - Containerized environments

1. Clone the Repository

git clone https://github.com/edgeof8/PyRC.git
cd PyRC
git checkout -b feature/your-feature-name

2. Set Up Python Environment

Using venv (Recommended)

python -m venv venv
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate
pip install -r requirements.txt
pip install -e .

Using Poetry

pip install poetry
poetry install

3. Configuration Setup

cp pyterm_irc_config.ini.example pyterm_irc_config.ini

Edit pyterm_irc_config.ini to customize your settings.

Running the Application

python pyrc.py

Or if installed:

pyrc

Running Tests

pytest
pytest tests/test_core.py
pytest --cov=pyrc_core
Tip: Use pre-commit install to set up Git hooks for automated linting.
