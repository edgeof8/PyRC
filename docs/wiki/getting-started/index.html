Getting Started with PyRC

Introduction

PyRC is a modern, asynchronous IRC client built with Python, designed to be both powerful and extensible. This guide will help you get started with PyRC, from installation to your first connection.

Note: PyRC requires Python 3.9 or higher.

Installation

You can install PyRC using pip:

# Clone the repository
git clone https://github.com/edgeof8/PyRC.git
cd PyRC

# Create and activate a virtual environment
python -m venv venv
# On Windows:
.\venv\Scripts\activate
# On Unix/macOS:
# source venv/bin/activate

# Install the package in development mode
pip install -e .

Configuration

PyRC looks for a configuration file at ~/.config/pyrc/config.ini or you can specify a custom path using the --config flag.

Here's a basic configuration example:

[server]
address = irc.libera.chat
port = 6697
ssl = true
nick = YourNickname
username = YourUsername
realname = Your Real Name
channels = #python,#testing

[ui]
theme = default
message_history = 1000

[logging]
level = INFO
file = pyrc.log

First Steps

Once you've installed and configured PyRC, you can start the client:

# Start PyRC
python -m pyrc_core

Basic Commands

  • /connect - Connect to a server
  • /join #channel - Join a channel
  • /msg nickname - Start a private message
  • /help - Show help information
Tip: Press Tab for command and nickname completion.
