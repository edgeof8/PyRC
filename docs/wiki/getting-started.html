Getting Started with PyRC

This guide will walk you through the initial steps to get PyRC running and introduce you to its core concepts.

New to PyRC? Start with the Quick Start Guide for a rapid setup.

Installation

For detailed installation instructions, including options for installing via pip or from source, please refer to the Quick Start Guide or the Development Setup page if you plan to contribute.

Configuration Overview

PyRC's behavior is managed through the pyterm_irc_config.ini file. This file allows you to define server connections, user identity, UI preferences, logging settings, and feature toggles. For a comprehensive overview of all configurable options, see the Advanced Usage - Configuration Deep Dive section.

Basic Usage and Commands

Once PyRC is installed and configured, you can launch it. The client provides a rich set of built-in commands for interacting with IRC networks and managing the client's behavior. To explore available commands, type /help within the client. For specific command details, use /help .

Learn more about the command system on the Command System page.

Understanding the UI

PyRC features a responsive terminal UI built with Python's curses library. It includes a message panel, sidebar for channels and users, and a status bar. Familiarize yourself with UI navigation and features on the UI System page.

Extending PyRC with Scripts

PyRC is highly extensible through Python scripts. You can write custom scripts to automate tasks, add new commands, or integrate with external services. Discover more about scripting and the available API on the Script API Reference page.
