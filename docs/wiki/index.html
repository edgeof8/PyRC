Quick Install

Stable Release:

pip install pyrc

Specific Version:

pip install pyrc==1.0.0

Upgrade Existing:

pip install --upgrade pyrc

From Source:

pip install git+https://github.com/edgeof8/PyRC.git

Requirements:

  • Python 3.8+
  • asyncio
  • curses (or windows-curses on Windows)
  • pyfiglet (optional, for /ascii command)
