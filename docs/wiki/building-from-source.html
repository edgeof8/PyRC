Building from Source

If you want to build PyRC from source, follow these instructions.

Advanced: Building from source is only necessary if you want to modify the core client logic.

Prerequisites

  • Python 3.8 or higher
  • Git
  • pip (Python package installer)
  • build package
  • PyInstaller

Build Steps

1. Clone the repository and set up environment

git clone https://github.com/edgeof8/PyRC.git
cd PyRC
python -m venv venv
# Activate virtual environment
# On Windows: venv\Scripts\activate
# On macOS/Linux: source venv/bin/activate
pip install -r requirements.txt
pip install -U pip build pyinstaller

2. Run the build script

This script will build the Python package and create standalone executables in the dist/ directory.

python build.py

3. Test the build (recommended)

python test_build.py

Verifies package installation, executable functionality, and configuration handling.

Build Output

The build process creates these files in dist/:

  • Python packages:
    • pyrc-.tar.gz
    • pyrc--py3-none-any.whl
  • Standalone Executables:
    • Windows: pyrc--win64.exe
    • Linux: pyrc--linux
    • macOS: pyrc--macos
